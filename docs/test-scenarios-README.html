Тестовые сценарии 360° Feedback

Быстрый старт

  1. Откройте в браузере test-scenarios-prototype.html — объединённые сценарии по шагам (роль → экран → что заполнить), базовый URL и кнопки на основные маршруты.
  2. Полный текст приёмки и пошаговые сценарии — в TEST_SCENARIO_360.md (в репозитории; в браузере по клику может открыться скачивание — тогда смотрите файл в IDE).

Локально: npm run dev, сид данных — npm run db:seed (id цикла и оцениваемых смотрите в выводе консоли).

Если проводник долго открывает папку

Файл можно открыть без обхода по каталогу:

  • Win+R — вставьте путь в кавычках и Enter:
    "c:\Users\Elovi\OneDrive\Документы\AI\360-feedback-app\docs\test-scenarios-prototype.html"
  • PowerShell / терминал Cursor: Start-Process "…путь…\test-scenarios-prototype.html" (тот же полный путь).
  • В Cursor: откройте HTML в редакторе → Ctrl+Shift+P → Simple Browser и укажите этот файл (или перетащите вкладку в браузер, если поддерживается).
